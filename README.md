# FlatUi

### Installation

First step

```sh
$ git clone git@github.com:Amstergo/flatUi.git
```

Then install the dependencies and devDependencies.

```sh
$ cd flatUi
$ yarn install
```

#### Run project

Start WDS (webpack-dev-server)

```sh
$ yarn run start:dev
```

Open in your browser

```sh
http://localhost:9000/
```
